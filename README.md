# ionflow.xyz

무료 온라인 개발자 유틸리티 도구 모음

## 도구 목록

- **QR코드 생성기** - URL, 텍스트를 QR코드로 변환
- **WiFi QR코드** - WiFi 정보를 QR코드로 생성
- **명함 QR코드** - vCard 형식 연락처 QR코드 생성
- **JSON 포맷터** - JSON 검증 및 포맷팅
- **Base64 변환기** - 텍스트/이미지 Base64 인코딩/디코딩
- **해시 생성기** - MD5, SHA-1, SHA-256 해시 생성
- **비밀번호 생성기** - 안전한 랜덤 비밀번호 생성
- **UUID 생성기** - UUID v4 생성
- **색상 변환기** - HEX, RGB, HSL 변환

## 기술 스택

- HTML5 / CSS3 / JavaScript
- [Pretendard](https://github.com/orioncactus/pretendard) 폰트
- [qrcodejs](https://github.com/davidshimjs/qrcodejs) 라이브러리

## 배포

Cloudflare Pages를 통해 자동 배포됩니다.
