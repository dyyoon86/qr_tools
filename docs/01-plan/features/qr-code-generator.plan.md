## 1. Overview

### 1.1 Background
ionflow.xyz는 현재 바둑/체스 게임만 제공하는 정적 HTML/CSS/JS 사이트이다.
min-inter.co.kr 분석 결과 185+ 도구를 보유한 종합 유틸리티 플랫폼의 구조를 파악했으며,
ionflow.xyz를 동일한 종합 도구 플랫폼으로 확장하는 것이 목표이다.

**QR 코드 생성기를 첫 번째 유틸리티 도구로 구현**하여 플랫폼 확장의 기반을 마련한다.

### 1.2 Goals
- QR 코드 생성 (텍스트, URL, WiFi, vCard, 이메일, 전화번호, SMS)
- 커스텀 디자인 (색상, 로고 삽입, 스타일)
- 다운로드 (PNG, SVG, PDF)
- 일괄 생성 기능
- 모바일 반응형 UI
- AdSense 수익화 적용

### 1.3 Scope
| In Scope | Out of Scope |
|----------|-------------|
| QR 코드 생성 (7가지 타입) | QR 코드 스캔/리더 |
| 커스텀 디자인 옵션 | 서버 사이드 생성 |
| PNG/SVG 다운로드 | 유료 기능/구독 |
| 반응형 UI | 사용자 계정/로그인 |
| AdSense 통합 | 다국어 지원 (한국어만) |
| SEO 최적화 | 타 도구 동시 구현 |

## 2. Requirements

### 2.1 Functional Requirements

#### FR-01: QR 코드 타입 지원
| 타입 | 입력 필드 | 우선순위 |
|------|----------|:--------:|
| **URL** | URL 입력 | P0 |
| **텍스트** | 자유 텍스트 | P0 |
| **WiFi** | SSID, 비밀번호, 암호화방식 | P1 |
| **vCard** | 이름, 전화, 이메일, 회사, 주소 | P1 |
| **이메일** | 수신자, 제목, 본문 | P1 |
| **전화번호** | 전화번호 | P2 |
| **SMS** | 전화번호, 메시지 | P2 |

#### FR-02: 커스텀 디자인
- 전경색 / 배경색 선택 (컬러피커)
- QR 코드 스타일 (사각형, 둥근모서리, 도트)
- 중앙 로고 삽입 (이미지 업로드)
- 에러 정정 레벨 선택 (L, M, Q, H)
- 크기 조절 (128px ~ 1024px)

#### FR-03: 다운로드
- PNG 다운로드 (고해상도)
- SVG 다운로드 (벡터)
- 클립보드 복사

#### FR-04: 미리보기
- 실시간 QR 코드 미리보기 (입력 즉시 갱신)
- 생성 히스토리 (localStorage, 최근 10개)

#### FR-05: 일괄 생성
- CSV/텍스트 입력으로 여러 QR 코드 동시 생성
- ZIP 다운로드

### 2.2 Non-Functional Requirements

| 항목 | 요구사항 |
|------|---------|
| **성능** | QR 생성 < 100ms, 페이지 로딩 < 2초 |
| **호환성** | Chrome, Edge, Firefox, Safari (최근 2버전) |
| **반응형** | 모바일 320px ~ 데스크톱 1920px |
| **접근성** | WCAG 2.1 AA |
| **SEO** | Lighthouse SEO 90+ |
| **보안** | 모든 처리 클라이언트 사이드 (데이터 서버 전송 없음) |

## 3. Technical Approach

### 3.1 기술 스택 결정

**선택: 순수 HTML/CSS/JS (기존 프로젝트 호환)**

| 검토 옵션 | 장점 | 단점 | 결정 |
|-----------|------|------|------|
| Next.js 마이그레이션 | 컴포넌트 재사용, SSR/SSG | 기존 구조 전면 변경, 빌드 필요 | 향후 검토 |
| 순수 HTML/CSS/JS | 기존 구조 호환, 즉시 배포, 빌드 불필요 | 대규모 확장 시 관리 어려움 | **채택** |
| Vite + Vanilla | 모듈 번들링, 빠른 개발 | 빌드 스텝 필요 | 향후 검토 |

> 첫 도구이므로 기존 프로젝트 구조와 동일하게 정적 HTML로 시작.
> 도구 10개 이상 추가 시 Next.js 마이그레이션 재검토.

### 3.2 핵심 라이브러리

| 라이브러리 | 용도 | CDN 크기 | 라이선스 |
|-----------|------|---------|---------|
| **qrcode.js** (davidshimjs) | QR 코드 생성 | ~17KB (gzip) | MIT |
| **qr-code-styling** | 커스텀 디자인 QR | ~35KB (gzip) | MIT |
| **JSZip** | 일괄 다운로드 ZIP 생성 | ~10KB (gzip) | MIT/GPL |
| **FileSaver.js** | 파일 다운로드 | ~3KB (gzip) | MIT |

> `qr-code-styling`이 로고 삽입, 도트 스타일 등을 기본 지원하여 주력 라이브러리로 채택.

## 5. Implementation Plan

### Phase 1: 기본 기능 (3일)
1. HTML 구조 + CSS 스타일링
2. URL/텍스트 QR 생성 (qr-code-styling 연동)
3. 실시간 미리보기
4. PNG/SVG 다운로드
5. 반응형 레이아웃

### Phase 2: 확장 타입 (2일)
1. WiFi QR 생성 폼
2. vCard QR 생성 폼
3. 이메일/전화/SMS QR 생성 폼
4. 타입별 입력 검증

### Phase 3: 커스텀 디자인 (2일)
1. 컬러 피커 (전경/배경)
2. QR 스타일 옵션 (사각/둥근/도트)
3. 로고 삽입 기능
4. 에러 정정 레벨 선택
5. 크기 조절 슬라이더

### Phase 4: 부가 기능 (2일)
1. 일괄 생성 (CSV 입력 + ZIP 다운로드)
2. 생성 히스토리 (localStorage)
3. 클립보드 복사
4. 다크 모드

### Phase 5: SEO + 수익화 (1일)
1. SEO 메타태그, Schema.org 마크업
2. AdSense 배너 배치
3. sitemap.xml 업데이트
4. Google Analytics 이벤트 추적

### 총 예상 기간: 10일
