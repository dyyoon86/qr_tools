Base64란?

Base64는 바이너리 데이터를 ASCII 문자열로 인코딩하는 방식입니다. 이메일 첨부파일, 데이터 URL, API 통신 등에서 널리 사용됩니다.

활용 사례

  • 이미지를 HTML/CSS에 직접 삽입
  • API로 바이너리 데이터 전송
  • 이메일 첨부파일 인코딩

특징

  • 크기가 약 33% 증가
  • A-Z, a-z, 0-9, +, / 문자 사용
  • 패딩에 = 사용
